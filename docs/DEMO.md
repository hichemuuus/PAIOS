# Veyron Demo Guide

## Quick Demo (5 minutes)
1. Launch Veyron desktop app
2. Dashboard shows system metrics (CPU, memory, processes)
3. Navigate to Agent workspace
4. Enter task: "What files are in my Documents folder?"
5. Watch the agent: intent classification → planning → tool execution → response
6. Check Memory Center to see the saved interaction
7. Check System Intelligence for live monitoring

## Deep Demo (15 minutes)
1. Show Dashboard — real-time sparklines, recent tasks, system overview
2. Agent Workspace — multi-step task, watch DAG planning
3. Task Registry — filter and inspect past tasks
4. Memory Center — search memory, inspect importance scoring
5. System Intelligence — CPU/memory/disk gauges, process list
6. Tool Center — inspect available tools and their schemas
7. Learning Dashboard — skill detection, workflow suggestions
8. Diagnostics — run health checks, inspect request history

## Example Tasks
- "Summarize the Python files in the current directory"
- "Check my system memory usage"
- "What tasks did I run yesterday?"
- "Create a report of my most used tools"
- "Monitor CPU usage and alert if above 80%"
