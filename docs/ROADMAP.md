# Veyron — Roadmap

## Completed
- Agent runtime with ReAct loop
- DAG planning engine with adaptive re-planning
- Hybrid memory system (keyword + importance + recency)
- 5 intelligence micro-models (intent, tool, memory, error, planning)
- Training pipeline v2 with benchmarks
- Model registry with promotion/rollback
- Desktop application (Tauri v2)
- System monitoring with WebSocket push
- Plugin SDK
- Workflow engine
- Comprehensive test suite (880+ tests)
- Production installer (NSIS/MSI)
- Auto-update system
- Security layer (path, command, safety policies)

## Future
- **Plugin Ecosystem** — Community plugin marketplace, plugin versioning, dependency management
- **Distributed Agents** — Multi-machine task orchestration
- **Advanced Planning** — Learning-based plan optimization, plan templates
- **Better Model Integration** — OpenAI/Anthropic API support, local fine-tuning
- **Frontend Tests** — Vitest unit tests, Playwright E2E tests
- **Multi-User Support** — User profiles, permissions, workspaces
- **Advanced Memory** — Vector embeddings, semantic search, summary-based retrieval
- **CI/CD Enhancements** — Automated test runs, artifact publishing
- **Linux/macOS Support** — First-class desktop support for all platforms
- **Performance Optimization** — Binary size reduction, startup time improvement
